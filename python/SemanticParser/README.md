## 语法制导翻译和语义分析

1. 每个文法符号都有一个属性集合（考虑为symbol 增加属性）
    属性可以是 综合属性 和/或 继承属性
2. 语法制导定义中 每个产生式都有一个对应的语义规则
    b = f(c1, c2, c3, c4, c5, ..., ck), 这里的 f 是一个函数
    1. 如果 b 是 A 的一个综合属性， 则c1,...,ck是产生式右部文法符号的属性或 A 的继承属性。
    2. 如果 b 是产生式右部某个符号的一个继承属性， 则 c1,...,ck 是 A 或产生式右部任何文法符号属性

* 左部符号的综合属性由右部符号计算的到
* 产生式右部的继承属性是由 所在产生式的 左部 或右部符号中继承出来的
